# HaramBlur

HaramBlur is an Android application designed to help users maintain focus and avoid distractions by detecting and blurring inappropriate content and providing tools to lock distracting apps when needed.

## Features

- **Content Detection & Blurring**: Automatically detects and blurs potentially inappropriate content
- **CODE RED Emergency Lock**: Instantly lock distracting apps when you need to focus
- **Statistics & Tracking**: Monitor your protection history and view detailed stats
- **Customizable Settings**: Adjust sensitivity levels and protection preferences
- **Islamic Reminders**: Receive helpful reminders throughout the day

## Technical Details

- Built with Jetpack Compose for modern UI development
- Material Design 3 implementation for a professional look and feel
- Responsive layouts for both phone and tablet form factors
- Dark mode support with optimized UI for both themes
- Edge-to-edge design with proper system bar handling
