## 📱 About HaramBlur

HaramBlur is a comprehensive Islamic-focused privacy application designed to help Muslims maintain digital purity and spiritual well-being in our modern connected world. Built with Islamic values at its core, the app provides real-time content filtering, website blocking, and emergency app locking capabilities - all while operating completely offline to ensure your privacy.

### 🎯 Mission Statement

*"Helping Muslims navigate the digital world while preserving their faith, privacy, and spiritual well-being."*

## ✨ Key Features

### 🛡️ Real-time Content Protection
- **Screen Content Monitoring**: Advanced detection of inappropriate visual content
- **Intelligent Blur System**: Multiple blur types including Islamic geometric patterns
- **Contextual Filtering**: Adaptive sensitivity based on content type and context
- **Performance Optimized**: Efficient processing with minimal battery impact

### 🚨 CODE RED Emergency System
- **Instant App Locking**: Lock distracting apps immediately when temptation strikes
- **Password Protection**: Secure session management with strong password requirements
- **Duration Control**: Flexible locking periods from minutes to days
- **Islamic Motivation**: Inspiring reminders during locked sessions

### 🕌 Islamic Content Integration
- **Authentic Quranic Verses**: Carefully selected verses about lowering gaze and purity
- **Hadith Collections**: Authentic sayings of Prophet Muhammad (PBUH)
- **Contextual Reminders**: Relevant Islamic guidance based on user behavior
- **Arabic Typography**: Beautiful Arabic text rendering with proper RTL support

### ⚙️ Comprehensive Settings
- **Detection Sensitivity**: Customizable content detection thresholds
- **Blur Preferences**: Choose from Gaussian blur, black box, or Islamic patterns
- **Performance Modes**: Optimize for battery life or maximum protection
- **Privacy Controls**: Complete control over data retention and storage

### 📊 Privacy-First Analytics
- **Local Statistics Only**: All data stays on your device
- **Progress Tracking**: Monitor your digital wellness journey
- **Achievement System**: Celebrate milestones in maintaining digital purity
- **No Data Collection**: Zero telemetry, analytics, or tracking

## 🏗️ Technical Architecture

### Modern Android Development
- **Jetpack Compose**: Modern, declarative UI framework
- **Clean Architecture**: Separation of concerns with testable code
- **MVVM Pattern**: Reactive programming with StateFlow
- **Dependency Injection**: Hilt for scalable architecture

### Database & Storage
- **Room Database**: Efficient local storage with SQLCipher encryption
- **DataStore**: Modern preferences management
- **Offline-First**: Complete functionality without internet connection
- **Encrypted Storage**: All sensitive data protected with industry-standard encryption

### Background Services
- **Foreground Service**: Continuous content monitoring
- **Accessibility Service**: System-level app control for CODE RED
- **Permission Management**: Comprehensive handling of Android permissions
- **Service Lifecycle**: Proper resource management and cleanup

## 📥 Installation

### Requirements
- **Android Version**: 8.0+ (API 26)
- **RAM**: 2GB minimum, 4GB recommended
- **Storage**: 100MB free space
- **Permissions**: Screen capture, Accessibility service (optional)

### Download & Install

1. **Download APK**: Get the latest version from [Releases](../../releases/latest)
2. **Enable Installation**: Allow installation from unknown sources in Android settings
3. **Install**: Tap the APK file and follow installation prompts
4. **Setup**: Launch app and complete initial setup with permission grants

### First Run Setup

1. **Welcome & Introduction**: Learn about app features and Islamic principles
2. **Permission Requests**: Grant necessary permissions with clear explanations
3. **Initial Configuration**: Set detection sensitivity and preferences
4. **Islamic Content Setup**: Configure reminder frequency and language preferences
5. **Ready to Use**: Start protecting your digital privacy immediately
